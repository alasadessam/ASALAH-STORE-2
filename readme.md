## 📋 نظرة عامة

**أصالة** هو متجر إلكتروني فاخر ومتكامل تم تطويره باستخدام تقنيات الويب الحديثة (HTML, CSS, JavaScript). يتميز المتجر بتصميم عصري وأنيق يتوافق مع جميع أحجام الشاشات، مع تجربة مستخدم سلسة وممتعة.

### ✨ المميزات الرئيسية

- 🎨 **تصميم فاخر وعصري** - واجهة مستخدم أنيقة بألوان متناسقة
- 📱 **تجاوب كامل** - يعمل على جميع الأجهزة (موبايل، تابلت، ديسكتوب)
- 🛒 **سلة تسوق متكاملة** - إضافة، تعديل، وحذف المنتجات
- 💳 **نظام دفع متكامل** - خطوات دفع واضحة مع تأكيد الطلب
- 🔍 **بحث متقدم** - بحث فوري عن المنتجات
- ❤️ **قائمة المفضلة** - حفظ المنتجات المفضلة
- ⭐ **تقييمات المنتجات** - نظام تقييم بنجوم
- 🏷️ **تصنيفات متنوعة** - منتجات جديدة، تخفيضات، الأكثر مبيعاً
- ⏰ **عداد تنازلي** - للعروض المحدودة
- 🔔 **إشعارات تفاعلية** - تنبيهات فورية للمستخدم
- 🌙 **تأثيرات بصرية** - حركات وانتقالات سلسة

## 🚀 التقنيات المستخدمة

### Frontend
- **HTML5** - هيكل الصفحة الدلالي

- **JavaScript (ES6+)** - الوظائف التفاعلية

- Mobile-First Approach
